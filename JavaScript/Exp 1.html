JAVASCRIPT MATH FUNCTIONS

Some of the math functions are showcased below

1. Absolute value is the distance between the number and zero: Eg, -24.34:

2. Ceil function returns the smallest integer that is greater than or equal to the number: Eg, 9.4:

3. Floor function rounds down and returns the largest integer less than or equal to a given number: Eg, 4.7:

4. Round function rounds a number to a specified number of digits: Eg, 12.8974:

5. Power function A power function is a function where y = x ^n where n is any real constant number: Eg, 3,2:

6. Square root function is used to return the square root of a number: Eg, 64:

7. Trunc returns the integer part of number: Eg, 6.12:

7. Sign returns if the number is negative positive or null: Eg, -4:
